## System Overview

### What is this SaaS?

An AI-powered appointment booking and lead recovery system for local businesses (dentists, med spas, real estate agents) that:

- **Captures leads** from website forms, WhatsApp, email, Facebook Ads
- **Auto-replies instantly** with AI chatbot (natural conversations)
- **Books appointments** through conversational UI
- **Sends reminders** via SMS/email (24h and 2h before)
- **Recovers abandoned leads** with smart reactivation sequences
- **Analyzes performance** with analytics dashboard

### Core Problems Solved

| Problem | Solution |
|---------|----------|
| 30-40% lead loss | AI chatbot never sleeps, captures leads 24/7 |
| Manual booking is slow | Conversational booking takes 30 seconds |
| High no-show rate | Automated reminders reduce no-shows by 20-30% |
| Lost follow-up | AI identifies abandoned leads, sends reactivation |
| No visibility into leads | Dashboard shows conversion funnel, revenue impact |

### Target Users

- **Dentists**: 200K+ practices in US, avg spending $1000-2000/month on ads
- **Med spas**: 30K+ locations, higher AD spend ($2000-5000/month)
- **Real estate agents**: 2M+ agents losing leads to competitors
- **Service businesses**: Salons, fitness studios, clinics

```javascript
class AuthService {
  // Signup
  async signup(email, password, businessName) {
    // Validate input
    if (!email.match(/^[\w\.-]+@[\w\.-]+\.\w+$/)) {
      throw new Error("Invalid email");
    }
    
    // Check if user exists
    const existingUser = await User.findOne({ email });
    if (existingUser) throw new Error("User already exists");
    
    // Hash password
    const hashedPassword = await bcrypt.hash(password, 10);
    
    // Create user
    const user = await User.create({
      email,
      password: hashedPassword,
      businessId: null,
      subscriptionTier: "trial"
    });
    
    // Create business
    const business = await Business.create({
      name: businessName,
      ownerId: user._id
    });
    
    // Link business to user
    user.businessId = business._id;
    await user.save();
    
    // Generate JWT
    const token = jwt.sign(
      { userId: user._id, businessId: business._id },
      process.env.JWT_SECRET,
      { expiresIn: "7d" }
    );
    
    return { user, business, token };
  }
  
  // Login
  async login(email, password) {
    const user = await User.findOne({ email });
    if (!user) throw new Error("User not found");
    
    const isValid = await bcrypt.compare(password, user.password);
    if (!isValid) throw new Error("Invalid password");
    
    const token = jwt.sign(
      { userId: user._id, businessId: user.businessId },
      process.env.JWT_SECRET,
      { expiresIn: "7d" }
    );
    
    user.lastLoginAt = new Date();
    await user.save();
    
    return { user, token };
  }
  
  // Verify Token
  verifyToken(token) {
    return jwt.verify(token, process.env.JWT_SECRET);
  }
}

module.exports = new AuthService();
```

#### 2. LeadService.js

```javascript
class LeadService {
  // Create lead from form
  async createLead(businessId, leadData) {
    const lead = await Lead.create({
      businessId,
      firstName: leadData.firstName,
      lastName: leadData.lastName,
      email: leadData.email,
      phone: leadData.phone,
      source: leadData.source || "website_form",
      sourceDetail: leadData.sourceDetail,
      status: "active",
      score: 10, // Initial score
      createdAt: new Date()
    });
    
    // Trigger welcome message
    await NotificationService.sendWelcomeMessage(businessId, lead);
    
    // Queue for AI chatbot response
    await chatQueue.add('engage_lead', { leadId: lead._id }, {
      delay: 1000,
      attempts: 3
    });
    
    return lead;
  }
  
  // Score lead based on actions
  async scoreLead(leadId, action) {
    const lead = await Lead.findById(leadId);
    const business = await Business.findById(lead.businessId);
    
    const scoreRules = business.leadScoring.rules;
    const points = scoreRules[action] || 0;
    
    lead.score += points;
    lead.scoreHistory.push({
      action,
      score: points,
      timestamp: new Date()
    });
    
    // Mark as abandoned if last contact > 7 days and score low
    if (
      lead.lastContactAt &&
      Date.now() - lead.lastContactAt > 7 * 24 * 60 * 60 * 1000 &&
      lead.score < 30
    ) {
      lead.isAbandoned = true;
      lead.abandonedAt = new Date();
      
      // Queue reactivation
      await reactivationQueue.add('send_reactivation', { leadId }, {
        delay: 60 * 1000
      });
    }
    
    await lead.save();
    return lead;
  }
  
  // Get conversion funnel
  async getConversionFunnel(businessId, dateRange) {
    const { startDate, endDate } = dateRange;
    
    const metrics = {
      totalLeads: await Lead.countDocuments({
        businessId,
        createdAt: { $gte: startDate, $lte: endDate }
      }),
      engaged: await Lead.countDocuments({
        businessId,
        status: "active",
        messageCount: { $gt: 0 },
        createdAt: { $gte: startDate, $lte: endDate }
      }),
      booked: await Lead.countDocuments({
        businessId,
        appointmentId: { $exists: true },
        createdAt: { $gte: startDate, $lte: endDate }
      }),
      completed: await Lead.countDocuments({
        businessId,
        appointmentId: { $exists: true },
        status: "converted",
        createdAt: { $gte: startDate, $lte: endDate }
      })
    };
    
    return {
      ...metrics,
      conversionRate: (metrics.booked / metrics.totalLeads * 100).toFixed(2),
      completionRate: (metrics.completed / metrics.booked * 100).toFixed(2)
    };
  }
}

module.exports = new LeadService();
```

#### 3. ChatbotService.js

```javascript
const Anthropic = require("@anthropic-ai/sdk");

class ChatbotService {
  constructor() {
    this.client = new Anthropic({
      apiKey: process.env.ANTHROPIC_API_KEY
    });
  }
  
  async generateResponse(businessId, leadId, userMessage) {
    const business = await Business.findById(businessId);
    const conversation = await Conversation.findOne({ leadId, businessId });
    const lead = await Lead.findById(leadId);
    
    // Build conversation history for context
    const messageHistory = conversation?.messages || [];
    const contextMessages = messageHistory.map(msg => ({
      role: msg.sender === "chatbot" ? "assistant" : "user",
      content: msg.content
    }));
    
    // Build system prompt with business context
    const systemPrompt = `You are a friendly booking assistant for ${business.name}.

Business Info:
- Type: ${business.industry}
- Services: ${business.services.map(s => s.name).join(", ")}
- Hours: ${JSON.stringify(business.settings.workHours)}
- Appointment Duration: ${business.settings.appointmentDuration} minutes

Lead Info:
- Name: ${lead.firstName} ${lead.lastName}
- Phone: ${lead.phone}
- Email: ${lead.email}
- Interested Service: ${lead.interestedService || "Not specified"}
- Preferred Time: ${lead.interestedTime || "Any time"}

Your goals:
1. Be warm, friendly, and professional
2. Understand what service they need
3. Ask about their preferred appointment date/time
4. Collect their information (already have some)
5. Confirm the booking
6. Never make up information about services/hours
7. If you can't help, offer to connect them with the team

Current Message from Lead: "${userMessage}"

Available appointment slots for the next 7 days:
${await this.getAvailableSlots(businessId)}

Respond naturally in 1-2 sentences unless explaining something. Try to move toward booking.`;

    try {
      const response = await this.client.messages.create({
        model: "claude-3-5-sonnet-20241022",
        max_tokens: 300,
        system: systemPrompt,
        messages: [
          ...contextMessages,
          { role: "user", content: userMessage }
        ]
      });
      
      const assistantMessage = response.content[0].text;
      
      // Save to conversation
      await Conversation.findByIdAndUpdate(conversation._id, {
        $push: {
          messages: [
            {
              sender: "lead",
              content: userMessage,
              timestamp: new Date(),
              type: "text"
            },
            {
              sender: "chatbot",
              senderName: business.chatbotConfig.name,
              content: assistantMessage,
              timestamp: new Date(),
              type: "text",
              isAiGenerated: true,
              confidence: response.stop_reason === "end_turn" ? 0.95 : 0.85,
              tokensUsed: response.usage.output_tokens
            }
          ]
        },
        updatedAt: new Date()
      });
      
      // Score the lead for engagement
      await LeadService.scoreLead(leadId, "chatbot_engaged");
      
      // Check if we should extract appointment
      const appointmentData = await this.extractAppointmentData(
        assistantMessage,
        business
      );
      if (appointmentData) {
        await AppointmentService.createFromChatbot(
          businessId,
          leadId,
          appointmentData
        );
      }
      
      return {
        message: assistantMessage,
        appointmentExtracted: !!appointmentData
      };
      
    } catch (error) {
      console.error("Claude API Error:", error);
      
      // Fallback response
      return {
        message: business.chatbotConfig.fallbackResponse,
        appointmentExtracted: false
      };
    }
  }
  
  async getAvailableSlots(businessId) {
    const business = await Business.findById(businessId);
    const slots = [];
    
    for (let i = 0; i < 7; i++) {
      const date = new Date();
      date.setDate(date.getDate() + i);
      
      const dayName = date.toLocaleString("en-US", { weekday: "long" });
      const dayHours = business.settings.workHours[dayName.toLowerCase()];
      
      if (!dayHours) continue; // Closed this day
      
      // Get booked appointments for this day
      const booked = await Appointment.find({
        businessId,
        startTime: {
          $gte: new Date(date.setHours(0, 0, 0)),
          $lt: new Date(date.setHours(23, 59, 59))
        }
      });
      
      // Generate available times (30-min slots)
      const [openHour, openMin] = dayHours.open.split(":").map(Number);
      const [closeHour, closeMin] = dayHours.close.split(":").map(Number);
      
      for (let hour = openHour; hour < closeHour; hour++) {
        for (let min = 0; min < 60; min += 30) {
          const slotTime = `${hour.toString().padStart(2, "0")}:${min.toString().padStart(2, "0")}`;
          
          // Check if slot is booked
          const isBooked = booked.some(apt =>
            apt.startTime.toLocaleString("en-US", {
              hour: "2-digit",
              minute: "2-digit",
              hour12: false
            }) === slotTime
          );
          
          if (!isBooked) {
            slots.push(`${dayName} ${slotTime}`);
          }
        }
      }
    }
    
    return slots.slice(0, 5).join(", "); // Return first 5 available
  }
  
  async extractAppointmentData(chatbotMessage, business) {
    // Simple extraction logic
    // In production, use NER or intent classification
    const hasConfirmation = /confirm|perfect|great|book|schedule/i.test(chatbotMessage);
    
    if (!hasConfirmation) return null;
    
    // This would typically call Claude again to extract structured data
    // For now, return null - handle manually
    return null;
  }
}
```

```
System creates Lead record + Conversation record

Claude AI extracts intent and responds with questions

    ↓
User provides date/time preferences
    ↓
Claude AI confirms and creates Appointment
    ↓
System sends confirmation email + calendar invite
```

```javascript
// chatbot.routes.js

router.post("/message", async (req, res) => {
  try {
    const { businessId, leadId, message } = req.body;
    
    // Validate input
    if (!message || message.trim().length === 0) {
      return res.status(400).json({ error: "Message cannot be empty" });
    }
    
    // Get business and lead
    const business = await Business.findById(businessId);
    const lead = await Lead.findById(leadId);
    if (!business || !lead) {
      return res.status(404).json({ error: "Business or lead not found" });
    }
    
    // Get or create conversation
    let conversation = await Conversation.findOne({ businessId, leadId });
    if (!conversation) {
      conversation = await Conversation.create({
        businessId,
        leadId,
        conversationType: "chatbot",
        messages: []
      });
    }
    
    // Add user message to conversation
    conversation.messages.push({
      sender: "lead",
      senderName: lead.firstName,
      content: message,
      timestamp: new Date(),
      type: "text"
    });
    
    // Call Claude API
    const systemPrompt = buildSystemPrompt(business, lead);
    
    const response = await anthropicClient.messages.create({
      model: "claude-3-5-sonnet-20241022",
      max_tokens: 300,
      system: systemPrompt,
      messages: conversation.messages
        .filter(m => m.timestamp) // Ensure valid messages
        .map(m => ({
          role: m.sender === "chatbot" ? "assistant" : "user",
          content: m.content
        }))
    });
    
    const assistantMessage = response.content[0].text;
    
    // Add assistant message to conversation
    conversation.messages.push({
      sender: "chatbot",
      senderName: business.chatbotConfig.name,
      content: assistantMessage,
      timestamp: new Date(),
      type: "text",
      isAiGenerated: true,
      confidence: 0.95,
      tokensUsed: response.usage.output_tokens
    });
    
    await conversation.save();
    
    // Score the lead
    await LeadService.scoreLead(leadId, "chatbot_engaged");
    
    // Try to extract appointment data
    const appointmentData = await extractAppointmentIntent(
      assistantMessage,
      message,
      business
    );
    
    if (appointmentData) {
      // Create appointment
      const appointment = await AppointmentService.createFromChatbot(
        businessId,
        leadId,
        appointmentData
      );
      
      return res.json({
        message: assistantMessage,
        appointmentCreated: true,
        appointment
      });

    res.json({
      message: assistantMessage,
      appointmentCreated: false
    });
    
  } catch (error) {
    console.error("Chatbot error:", error);
```

### Lead Recovery/Reactivation Queue

```javascript
// reactivationQueue.js using Bull

const Queue = require("bull");
const reactivationQueue = new Queue("lead-reactivation", process.env.REDIS_URL);

// Process reactivation jobs
reactivationQueue.process(async (job) => {
  const { leadId } = job.data;
  
  const lead = await Lead.findById(leadId);
  const business = await Business.findById(lead.businessId);
  
  if (!lead || lead.status === "converted") return;
  
  // Check if already attempted recently
  const lastAttempt = lead.recoveryAttempts[lead.recoveryAttempts.length - 1];
  if (lastAttempt && Date.now() - lastAttempt.sentAt < 24 * 60 * 60 * 1000) {
    return; // Already tried in last 24 hours
  }
  
  // Determine message based on lead data
  let message = "";
  if (lead.interestedService) {
    message = `Hi ${lead.firstName}, still interested in ${lead.interestedService}? We have availability next week!`;
  } else {
    message = `Hi ${lead.firstName}, we'd love to help! Check out our services and schedule a time that works for you.`;
  }
  
  // Send via preferred method
  if (lead.preferredContact === "email") {
    await NotificationService.sendEmail(lead.email, {
      subject: `${business.name} - Let's get you scheduled`,
      body: message,
      callToAction: "View Availability"
    });
  } else {
    await NotificationService.sendSMS(lead.phone, message);
  }
  
  // Log recovery attempt
  lead.recoveryAttempts.push({
    type: lead.preferredContact,
    message,
    sentAt: new Date(),
    opened: false,
    clicked: false
  });
  
  lead.lastReactivationAt = new Date();
  await lead.save();
  
  // Schedule follow-up if still no response (7 days later)
  if (lead.recoveryAttempts.length < 3) {
    await reactivationQueue.add({ leadId }, {
      delay: 7 * 24 * 60 * 60 * 1000, // 7 days
      attempts: 2
    });
  }
});
```

```javascript
class GoogleCalendarService {
  async getAuthUrl(businessId) {
    const oauth2Client = new google.auth.OAuth2(
      process.env.GOOGLE_CLIENT_ID,
      process.env.GOOGLE_CLIENT_SECRET,
      process.env.GOOGLE_REDIRECT_URI
    );
    
    const scopes = [

      "https://www.googleapis.com/auth/calendar.readonly"
    ];
    
    const authUrl = oauth2Client.generateAuthUrl({
      access_type: "offline",
      scope: scopes,
      state: businessId // Pass business ID in state
    });
    
    return authUrl;
  }
  
  async handleCallback(businessId, authCode) {
    const oauth2Client = new google.auth.OAuth2(
      process.env.GOOGLE_CLIENT_ID,
      process.env.GOOGLE_CLIENT_SECRET,
      process.env.GOOGLE_REDIRECT_URI
    );
    
    const { tokens } = await oauth2Client.getToken(authCode);
    
    // Store encrypted tokens
    const integration = await Integration.findOneAndUpdate(
      { businessId, service: "google_calendar" },

        credentials: {
          accessToken: encrypt(tokens.access_token),

        },
        config: { calendarId: "primary" },
        status: "connected",
        connectedAt: new Date()
      },

      process.env.GOOGLE_CLIENT_SECRET,
      process.env.GOOGLE_REDIRECT_URI
    );
    
    oauth2Client.setCredentials({
      access_token: decrypt(integration.credentials.accessToken),
      refresh_token: decrypt(integration.credentials.refreshToken)

      summary: `${appointment.serviceName} - ${lead.firstName} ${lead.lastName}`,
      description: appointment.notes,
      start: {
        dateTime: appointment.startTime,
        timeZone: business.settings.timezone
      },
      end: {
        dateTime: appointment.endTime,
        timeZone: business.settings.timezone
      },
      attendees: [
        {
          email: lead.email,
          displayName: `${lead.firstName} ${lead.lastName}`
        }
      ],
      reminders: {
        useDefault: false,
        overrides: [
          { method: "email", minutes: 24 * 60 },
          { method: "popup", minutes: 15 }
        ]
      }
    };
    
    // Create or update event
    if (appointment.googleCalendarEventId) {
      await calendar.events.update({
        calendarId: integration.config.calendarId,
        eventId: appointment.googleCalendarEventId,
        resource: event
      });
    } else {
      const result = await calendar.events.insert({
        calendarId: integration.config.calendarId,
        resource: event
      });
      
      appointment.googleCalendarEventId = result.data.id;
      await appointment.save();
    }
    
    return event;
  }
}

module.exports = new GoogleCalendarService();
```

### Twilio SMS Integration

```javascript
// services/twilio.js

const twilio = require("twilio");

class TwilioService {
  constructor() {
    this.client = twilio(
      process.env.TWILIO_ACCOUNT_SID,
      process.env.TWILIO_AUTH_TOKEN
    );
  }
  
  async sendSMS(businessId, toPhone, message) {
    const business = await Business.findById(businessId);
    const integration = await Integration.findOne({
      businessId,
      service: "twilio",
      status: "connected"
    });
    
    if (!integration) throw new Error("Twilio not configured");
    
    try {
      const result = await this.client.messages.create({
        body: message,
        from: integration.config.phoneNumber,
        to: toPhone
      });
      
      return {
        sid: result.sid,
        status: result.status,
        sentAt: new Date()
      };
    } catch (error) {
      console.error("SMS error:", error);
      throw error;
    }
  }
  
  async sendAppointmentReminder(appointmentId) {
    const appointment = await Appointment.findById(appointmentId);
    const lead = await Lead.findById(appointment.leadId);
    const business = await Business.findById(appointment.businessId);
    
    const message = `Hi ${lead.firstName}, reminder: you have a ${appointment.serviceName} appointment tomorrow at ${appointment.startTime.toLocaleTimeString([], { hour: '2-digit', minute: '2-digit' })} at ${business.name}. Reply CONFIRM to confirm or CANCEL to cancel.`;
    
    return this.sendSMS(
      appointment.businessId,
      lead.phone,
      message
    );
  }
}

module.exports = new TwilioService();
```

## End-to-End Flow Diagram

```
1. LEAD CAPTURE
   Customer lands on website
   ↓
   Sees embedded form widget
   ↓
   Fills: First name, last name, email, phone, service
   ↓
   System creates Lead + Conversation (MongoDB)

2. AI ENGAGEMENT
   Welcome email sent (SendGrid)
   ↓
   Chatbot triggers with greeting
   ↓
   Lead: "I want teeth cleaning on Tuesday morning"
   ↓
   Claude AI processes and responds
   ↓
   Lead: "What time do you have?"
   ↓
   Claude suggests 3 available slots
   ↓
   Lead: "10 AM works!"

3. BOOKING
   Claude confirms all details
   ↓
   System creates Appointment record
   ↓
   Appointment syncs to Google Calendar
   ↓
   Confirmation email sent to lead + dentist
   ↓
   Lead score updated to "booked"

4. REMINDERS
   24h before: Email reminder sent
   ↓
   2h before: SMS reminder sent
   ↓
   Lead confirms via SMS reply

5. ANALYTICS
   Dashboard shows:
   ├─ 1 lead captured
   ├─ 3 chatbot messages
   ├─ 1 appointment booked
   ├─ $150 revenue generated
   └─ 5 min from capture to booking

6. FOLLOW-UP (if no-show or abandoned)
   If lead doesn't show up:
   ├─ Mark appointment as "no_show"
   ├─ Queue reactivation email (24h later)
   ├─ Email: "We missed you! Reschedule here"
   ├─ If no response in 7 days, send SMS
   └─ Track recovery attempts

7. LEAD RECOVERY (Abandoned Leads)
   If lead clicks form but doesn't book:
   ├─ Lead scores < 30
   ├─ Last contact > 7 days ago
   ├─ Queue reactivation sequence
   ├─ Send email: "Still interested in teeth cleaning?"
   ├─ If clicked, restart chatbot
   └─ If no response in 7 days, send SMS
```

## API Authentication Flow

```
Client Request
    ↓
1. Client sends credentials (email + password)
2. Server validates against MongoDB
3. Server generates JWT token (expires in 7 days)
4. Client stores token in localStorage
    ↓
Authenticated Request
    ↓
1. Client includes token in Authorization header
2. Middleware verifies token signature
3. Extract userId + businessId from token
4. Allow/deny based on businessId match
    ↓
If Token Expired
    ↓
1. Client sends refresh token
2. Server validates refresh token
3. Issues new JWT token
4. Client continues with new token
```
